# pitrs_towing


**Features**

* Vehicle towing using a physical rope system

**Two rope types:**

* Standard rope – lower durability

* Steel rope – higher durability

* Configurable rope break speed

* Real-time rope break detection based on vehicle speed

* Optimized for smooth performance

* OX Target interaction support

* Full OX Lib integration

* OX Inventory support (items + installation included)

* ESX framework support

**Configuration**

* Set rope break speed in config.lua

* Configure durability for each rope type

* Customize item usage for OX Inventory
